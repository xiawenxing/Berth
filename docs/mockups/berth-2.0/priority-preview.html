优先级样式预览 — 左色条 vs 标签

同一组任务（P0 / P1 / P1 / P2）在三种方案下的样子。每张卡都带 可点击的优先级标签（点击弹出 P0/P1/P2 切换，A 方案已画出弹层示意）。

A · 弱化色条 + 标签推荐
只有 P0 红条醒目；P1 橙降到 50%、更细；P2 极淡。色条做「环境扫视」，footer 标签做「明示+点击改」。
