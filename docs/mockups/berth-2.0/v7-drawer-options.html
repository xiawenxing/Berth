会话抽屉样式 · 二选一

点击一个会话，从右侧滑入一个抽屉覆盖在项目页之上；项目页留在左侧（变暗）， 可以直接点另一个会话切换，× / Esc / 点暗区都能关。抽屉占满全高：主体 = 会话 / 终端内容（等宽转写 + 闪烁光标）， 底部 = 输入对话框。下面并排展示两种抽屉顶部样式，便于横向比较。

区别只在抽屉顶部：样式 1 顶部是一张会话信息卡片（cli 标签 + 标题 + cwd + 在航 + 航线 chip + 行内操作），下面接分隔线 → 转写 → 输入框；样式 2 顶部只有一条瘦头栏（cli · 标题 · cwd · 在航 + ■停止 + ×关闭），下面直接是转写 → 输入框。转写与输入框两者完全一致。
1 带卡片
抽屉顶部是一张会话信息卡片：cli 标签 + 标题（13px/600）+ cwd + 在航状态 + 关联的航线/任务 chip + 悬停浮现的行内操作（pin / 改名 / ⋯）。卡片下接细分隔线，再是转写，再是底部输入框。信息密度高、上下文齐全。
