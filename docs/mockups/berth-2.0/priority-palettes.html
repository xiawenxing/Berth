每列是一个 P1-为主的看板（1 张进行中 + 4 张 P1 + 1 张罕见 P0 + 1 张 P2）——重点看：整列 P1 是否平静、那张 P0 能不能跳出来。色条已收细到 2px；每张卡的 P0/P1/P2 标签可点击改优先级。右上角切日/夜。

精炼语义
P1 用低饱和的暖驼色琥珀（而非醒目金黄）做 2px 细条与柔和暖灰琥珀 chip，单卡饱和度被刻意压低，整列 P1 只读作环境质感而非噪音；P0 保留满饱和红色警示、P2 退到近中性灰蓝，三者权重 P0≫P1>P2，确保 P1 多卡看板始终平静。
